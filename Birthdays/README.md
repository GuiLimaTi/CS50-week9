# 🎂 Birthdays Web Application

## 📌 Overview
This project is part of Harvard's CS50 Week 9 (Flask).
It is a simple web application that allows users to store and manage birthdays using a SQLite database.

The application demonstrates the integration between a backend built with Flask and a frontend using HTML and CSS, including server-side validation and database operations.

---

## 🚀 Features
- Add new birthdays through a web form
- Store data in a SQLite database
- Display all stored birthdays dynamically
- Server-side validation for user input
- Clean and structured Flask project architecture

---

## 📊 Functionality
- Users can submit a name, month, and day
- Data is validated on the server before insertion
- Valid entries are stored in the database
- All stored birthdays are displayed in a table
