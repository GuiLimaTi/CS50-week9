# 💸 CS50 Finance

A full-stack web application that simulates a stock trading platform, allowing users to manage a virtual portfolio by buying and selling stocks in real time.

---

## 🚀 Overview

CS50 Finance is a web application built with **Python (Flask)** and **SQLite**, designed to replicate the core functionalities of a stock trading system.

Users can create an account, search for stock prices, execute transactions, and track their financial portfolio dynamically.

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Database:** SQLite  
- **Frontend:** HTML, CSS (Bootstrap)  
- **Templating:** Jinja  

---

## 📊 Core Features

- User authentication (register & login with hashed passwords)  
- Real-time stock price lookup via API  
- Buy and sell shares with full validation  
- Portfolio tracking with updated balances  
- Transaction history with timestamps  

---

## 🧠 Key Skills Demonstrated

- Full-stack web development using Flask  
- Secure authentication and session management  
- Relational database design with SQL queries  
- Input validation and error handling  
- Integration with external APIs for real-time data
