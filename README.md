# 🌐 CS50 Week 9 — Web Development with Flask

## 📌 Overview
This repository contains my work for Week 9 of Harvard's CS50: Introduction to Computer Science.

During this week, I explored web development using Flask, focusing on building dynamic web applications that integrate backend logic, databases, and user interaction.

The projects developed in this module demonstrate how to handle HTTP requests, manage user input, connect to databases, and render dynamic content using templates.

---

## 🧠 Key Concepts Learned

- Building web applications with Flask
- Understanding HTTP methods (GET vs POST)
- Handling forms and user input
- Using Jinja for dynamic HTML rendering
- Structuring Flask applications (templates & static folders)
- Working with SQLite databases
- Implementing server-side validation
- Managing user sessions and authentication (Finance project)

---

## 🚀 Projects

### 🎂 Birthdays
A simple web application that allows users to store and manage birthdays.

**Main Features:**
- Add new birthdays via form submission
- Store data in a SQLite database
- Display all stored entries dynamically
- Validate user input on the server side

**Tech Used:**
- Flask
- SQLite
- HTML & CSS

---

### 💰 Finance 
A full-stack web application simulating a stock trading platform.

**Main Features:**
- User registration and login system
- Real-time stock price lookup via API
- Buy and sell stocks
- Portfolio tracking
- Transaction history

**Tech Used:**
- Flask
- SQLite
- Jinja Templates
- External API integration
